# Лабораторная работа по предмету "Программирование в КС"
## Группа: ИВТ-24-1б

Реализация модели для детекции лиц людей.

Исходный датасет, на котором обучалась модель: [ссылка](https://universe.roboflow.com/novo-mgqqd/face-detection-yyxs8/dataset/1)

Файлы лабораторной работы:
- `best_model.pth` - веса обученной модели
- `model.py` — класс архитектуры модели
- `out.zip` - результаты тестирования обученной модели
- `sort_data.py` - скрипт конвертации датасета из исходного вида к удобному для работы
- `test.py` - скрипт тестирования модели на сторонних данных
- `train.ipynb` —  скрипт обучения модели

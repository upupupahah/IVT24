# Лабораторная работа по предмету "Программирование в КС"
## Группа: ИВТ-24-1б

Репозиторий содержит 2 папки с результатами выполнения заданий
- face_detection: детекция лиц людей
- human_segmentation: сегментация контура человека

Каждая папка содержит в себе:
- best_model.pt: веса обученной модели
- скрипт с преобразованиями исходных данных к требуемому
- out.zip: результаты тестирования модели
- model.py: архитектура модели
- train.ipynb: ноутбук с обучением модели
